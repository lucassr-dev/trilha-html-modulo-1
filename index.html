<!DOCTYPE>

A tag <!DOCTYPE> define o tipo de documento HTML. Ela deve ser a primeira linha do código HTML e ajuda os navegadores a interpretarem corretamente o documento.

<html>

A tag <html> é o elemento raiz de um documento HTML. Todos os outros elementos devem ser filhos diretos ou indiretos desta tag.

Voltar para o Índice

<head>

A tag <head> contém informações sobre o documento HTML, como metadados, títulos e links para arquivos CSS e JavaScript.

<title>

A tag <title> define o título do documento HTML. Ele é exibido na barra de título do navegador ou na guia do navegador.

<font>

A tag <font> é usada para alterar a cor, o tamanho e o estilo da fonte do texto. No entanto, ela não é mais recomendada e seu uso é considerado obsoleto. É preferível usar estilos CSS para estilizar o texto.

Voltar para o Índice

<del>

A tag <del> é usada para representar texto que foi removido ou excluído. O texto dentro desta tag geralmente é exibido com uma linha horizontal sobre ele.

Voltar para o Índice

<p>

A tag <p> é usada para definir um parágrafo de texto. Ela é usada para agrupar um bloco de texto em um parágrafo separado.

Voltar para o Índice

<abbr>

A tag <abbr> é usada para representar uma abreviação ou sigla. Ela pode ser usada para fornecer uma descrição completa da abreviação quando o mouse é passado sobre ela.
